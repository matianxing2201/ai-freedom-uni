# AI Freedom Home 项目分析报告

## 项目概述

这是一个基于 Flutter 开发的 AI 家居装修应用，主要功能包括：

- 用户登录/注册（支持手机号和邮箱）
- AI 装修任务（空房设计、现房改造、软装换搭）
- 个人中心管理
- 视频素材展示

**API 基础地址**: `https://api.aifreedomhome.com/`

```typescript
// 手机号验证码登录
interface MobileCodeLoginRequest {
  countryCode: string;   // 国家区号，如 "+86"
  mobile: string;        // 手机号
  captcha: string;       // 验证码
  loginType: 1;          // 1-手机号验证码登录
}

// 邮箱验证码登录
interface EmailCodeLoginRequest {
  email: string;         // 邮箱地址
  captcha: string;       // 验证码
  loginType: 2;          // 2-邮箱验证码登录
}

// 密码登录（手机号）
interface MobilePasswordLoginRequest {
  countryCode: string;   // 国家区号
  mobile: string;        // 手机号
  loginPassword: string; // 密码
  loginType: 0;          // 0-密码登录
}

// 密码登录（邮箱）
interface EmailPasswordLoginRequest {
  email: string;         // 邮箱地址
  loginPassword: string; // 密码
  loginType: 0;          // 0-密码登录
}

type LoginRequest = MobileCodeLoginRequest | EmailCodeLoginRequest | MobilePasswordLoginRequest | EmailPasswordLoginRequest;
```

**响应数据**:
```typescript
interface TaskCreateBean {
  taskType: number;  // 任务类型 0-空房设计 1-现房改造 2-软装换搭
  nums: number;      // 可创建数量
}

interface UserInfo {
  accessToken: string;           // 登录 token
  avatar: string;                // 头像URL
  nickname: string;              // 昵称
  profession: string;            // 职业
  uid: string;                   // 用户ID
  account: string;               // 账号
  countryCode: string;           // 国家区号
  mobile: string;                // 手机号
  email: string;                 // 邮箱
  age: number;                   // 年龄
  currentRoleId: number;         // 用户角色 1-AI自由家
  thirdLoginVO: boolean;         // 是否为第三方登录
  expiresIn: number;             // token 过期时间
  newUserFlag: boolean;          // 是否新用户
  sex: number;                   // 性别
  isNotice: number;              // 是否开启通知 0-开 1-关
  isPrivacy: number;             // 隐私设置
  hasPwd: number;                // 是否设置密码 0-未设置 1-已设置
  validEnd: string;              // 会员过期时间
  tokenInfoVO: Record<string, any>;
  taskCreateNums: TaskCreateBean[]; // 任务创建数量限制
}

interface LoginResponse {
  // 返回 UserInfo 结构
}
```

### 关键技术点

1. **状态管理**: 使用 Pinia 替代 GetX

4. **轮询机制**: 使用 `setInterval` 或 `useInterval` (VueUse)
5. **本地存储**: 使用 `localStorage` 替代 `sp_util`
